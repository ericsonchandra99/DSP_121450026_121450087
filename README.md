## ğŸ Cara Membuat Environment Python (Virtual Environment) 

### 1. Buat environment baru

```bash

```

### 2. Aktifkan environment

* Windows (PowerShell):

```powershell
.\venv\Scripts\Activate.ps1
```

* Windows (Command Prompt):

```cmd
venv\Scripts\activate.bat
```

* Linux/macOS:

```bash

```

```bash

```bash

```

```bash
